# 🪑 Furniture Store UI

A modern, responsive, and minimalist User Interface for an E-commerce Furniture Store. This project focuses on clean aesthetics and a seamless user experience to showcase premium furniture collections.

## 🪑 About the Project

Monogram is a high-end furniture brand UI concept designed for homes, hotels, and commercial spaces. The design is inspired by editorial fashion websites, using rich typography, cinematic imagery, and storytelling-driven section layouts to create an immersive shopping experience.



## 🚀 Features
* **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop views.
* **Product Catalog:** Clean grid layout for browsing furniture items.
* **Interactive Elements:** Smooth hover effects and transitions.
* **Modern Branding:** Professional typography and color palette.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Flexbox/Grid)
* **Design Style:** Minimalist / Modern
* **Version Control:** Git & GitHub

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Markup | HTML5 |
| Styling | CSS3 — Custom Properties, Flexbox, Grid, Animations |
| Logic | Vanilla JavaScript (ES6+) |
| Fonts | Google Fonts — Cormorant Garamond, Jost |
| Images | Unsplash (CDN) |
| Version Control | Git & GitHub |
